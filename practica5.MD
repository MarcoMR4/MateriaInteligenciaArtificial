### Desarrollo 
#### Percepci√≥n y an√°lisis del problema 
La primera impresi√≥n que tengo de este problema es que veo una imagen con muchos objetos distribuidos y que adem√°s hay una gran variedad de colores. La mayor√≠a de estos son alimentos que tienen tama√±os, formas y colores distintos. 
Lo primero que se me viene a la mente en esta situaci√≥n es que en un concepto muy b√°sico una imagen puede ser considerada como una matriz, una matriz de pixeles, a fin de cuentas, en donde cada uno tiene un color distinto. 
Otra caracter√≠stica que observo es que la imagen es una matriz muy grande que, de hecho, su tama√±o puede no siempre estar definido como lo es el caso de una matriz peque√±a, pero a fin de cuentas si tomamos las medidas del n√∫mero de filas y columnas entonces el ejercicio se puede resolver de manera muy similar al anterior, que es el de las islas, a fin de cuentas, en este caso es como si cont√°ramos islas, pero ahora manejando un color como referencia. 
Para tener una referencia m√°s clara y limitar los colores podr√≠amos utilizar los rangos num√©ricos en que los colores se acercan a los tonos rojos y tener una sola referencia para detectar todos los objetos que puedan considerarse ‚Äúrojos‚Äù. 
Los colores que normalmente hemos manejado en estos rangos son los RGB que son los tres colores primarios que forman a los dem√°s, aunque solo es un formato de muchos que se manejan para el color digital. 
El primer paso ser√≠a poder procesar la imagen para convertirla en una matriz, esto utilizando una librer√≠a que permita el manejo de im√°genes y colores como el caso de open cv del lenguaje Python, luego definir los colores. Adem√°s se deber√≠a definir un contador para guardar el n√∫mero de elementos, objetos o islas encontrados con la caracter√≠stica que se solicita. 
Ejemplo de algoritmo 

```python
r1 = 100, g1 = 0, b1 = 0
r2 = 255, g2 = 100, b2 = 100
islas = 0

mapa = cv.imread('atividad5.png', 1)  #leer los 3 canales con 1
w =mapa.shape[0]
h =mapa.shape[1]   #w y h para el ancho y alto de la imagen (filas y columnas)
#funcion para evaluar el color 
def rojo(i, j):
    if(mapa[i, j][2] >= r1 and mapa[i, j][2] <= r2):
        if(mapa[i, j][1] >= g1 and mapa[i, j][1] <= g2):
            if(mapa [i, j][0] >= b1 and mapa[i, j][0] <= b2):
                return True;
    return False;

M√©todo recursivo. 
def metodoRecursivo(i, j):
    if not(i < 0 or i >= w or j < 0 or j >= h):
        if(rojo(i,j)):
            mapa[i,j][2] = 0
            mapa[i,j][1] = 0
            mapa[i,j][0] = 0
            metodoRecursivo(i - 1, j)
            metodoRecursivo(i + 1, j)
            metodoRecursivo(i, j - 1)
            metodoRecursivo(i, j + 1)
    return

Ejemplo iterativo: 
for i in range(w):
    for j in range(h):
        if(rojo(i, j)):
            islas += 1
            metodoRecursivo(i, j);
Mostrar resultados 
cv.imshow('Original',mapa)
print('islas:',islas)
cv.waitKey(0)
cv.destroyAllWindows()
```
